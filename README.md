# Zaphyr Labs

## About

Zaphyr Labs is a personal development space focused on creating modern web projects and experimenting with interface design, interactions, and new ideas.

This is not a traditional portfolio. It is a working collection of projects and experiments, designed to evolve over time.

## Deployment

This project is deployed using GitHub Pages.

Steps to deploy:

1. Push all files to the main branch
2. Navigate to Settings → Pages
3. Select:

   * Branch: main
   * Folder: / (root)

The site will be available at:
https://zaphyrlabs.github.io

## Customization
